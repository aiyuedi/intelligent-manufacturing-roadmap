## 这是什么

智能制造工程（专业代码 080213T）是 2017-2018 年教育部新批的机械类本科专业。它位于**机械工程 × 控制科学 × 计算机科学**的交叉地带，目标是培养能设计、集成、优化智能产线和数字化工厂的复合型工程师。

计算机专业有 [developer-roadmap](https://github.com/kamranahmedse/developer-roadmap)（30万+ ⭐），前端/后端/DevOps 各自有清晰的学习路径。但智能制造工程——**没有**。这个 repo 就是为了补上这个空缺。

### 这份路线图包含

- ✅ 按 8 个学期的完整课程地图（含学分、优先级标注、前置依赖）
- ✅ 机械 → 控制 → 计算机 三轴技能矩阵
- ✅ 全部先修/后修课程依赖链（可视化 ASCII 图）
- ✅ 考研（控制科学与工程 081100）vs 就业（智能工厂方向）双线规划
- ✅ S5（大三上）决断标准：走考研还是走就业
- ✅ B站/教材/硬件/GitHub 仓库 资源索引
- ✅ 毕业时应该能做什么——可勾选的技能清单

### 适合谁看

- 智能制造工程大一/大二在读学生
- 机械/自动化专业想往智能制造方向转的学生
- 对智能工厂、工业互联网、数字孪生感兴趣的工科生

## 🎯 技能矩阵：三轴模型

智能制造工程师的核心能力不是单一专业的纵向深入，而是**三轴交叉处的横向整合能力**：

```
        🟦 机械工程轴
        │
    ┌───┼───┐
    │   │   │     ← 智能制造工程师 =
    │   │   │       能理解机械结构 +
    │   │   │       能设计控制逻辑 +
    └───┼───┘       能用软件实现自动化
        │
    🟩 控制/自动化轴          🟨 计算机/数据轴
```

### 三轴详解

| 轴 | 核心课程 | 毕业时应该能 | 对应工业岗位 |
|----|---------|-------------|-------------|
| 🟦 **机械工程** | 工程图学 1/2、工程力学、机械原理与设计、机械材料与制造、数控技术 | 看懂零件图/装配图，理解公差配合，知道常见传动机构 | 机械设计助理、工艺工程师 |
| 🟩 **控制/自动化** | 电工技术 → 电子技术 → 自动控制原理 → 单片机 → PLC → 传感检测 | 设计顺序控制逻辑，读写梯形图，选传感器型号，排查电路信号 | PLC工程师、自动化工程师、设备调试 |
| 🟨 **计算机/数据** | Python → 大数据与云计算 → 工业互联网 → 数据库 → 数字孪生 | 写脚本处理生产数据，理解 MES 数据流，会用 Git | MES实施顾问、数字化工程师 |

> **关键认知**：你不是要跟机械专业比画图、跟计算机专业比刷题、跟自动化专业比控制理论。你是三个都懂一点、能把它们串起来的人——这才是智能制造工程的价值。

## 🔗 先修后修关键链

### 数学链
```
高等数学A1(S1) → 高等数学A2(S2) → 复变函数与积分变换(S3) → 工程计算方法(S4)
线性代数B(S2) → 概率论与数理统计(S3) ────────────┘
线性代数B(S2) → 自动控制原理(S5) [状态空间需要矩阵运算]
```

### 物理·电工·电子链
```
大学物理B(S2) → 电工技术(S3) → 电子技术(S4) → 单片机原理与应用(S5)
                                          └→ 机电传动控制与PLC(S5)
电子认知实习(S1) → 物理实验B(S2) → 电工与电子技术实验(S4) → 电子工程训练1(S5)
```

### 图学·制造·设计链

## ⚖️ S5 决断标准

> S5（大三上）期末，根据以下量化标准做决断。**S5 之前不做二选一，两条线同时准备。**

| 条件 | 走向 |
|------|------|
| 自动控制原理 ≥ 85，数学类课均分 ≥ 80，心态扛得住 | → **考研为主**（路线A），就业备选 |
| 自动控制原理 < 80，或补修拖后腿，或经济压力变大 | → **就业为主**（路线B），考研试水或放弃 |
| 中间地带（75-84） | → **双线继续**，S6 再决断 |

**数学类课均分** = (高数A1 + 高数A2 + 线代B + 概率论 + 复变函数) / 5

> ⚠️ **S5 结束前不报名考研。** 成绩出来后（约2月初）再决定报不报。

## ✅ 毕业技能清单

> 毕业时逐条打勾。这些是你面试时能拿出手的东西。

### 🟦 机械

- [ ] 能看懂零件图和装配图，理解三视图投影原理
- [ ] 理解公差配合（基孔制/基轴制、公差等级）
- [ ] 知道常见传动机构：齿轮、链、带、丝杠、连杆、凸轮
- [ ] 了解常见加工工艺：车铣刨磨钻 + 数控加工

### 🟩 控制/自动化

- [ ] 会用万用表、示波器排查电路信号
- [ ] 能独立接一个简单电气控制柜（PLC + 继电器 + 接触器 + 传感器）
- [ ] 梯形图基本指令 + 功能指令熟练，能写顺序控制/报警处理/手动自动切换逻辑
- [ ] 理解 PID 控制原理，能用 Simulink 调参仿真
- [ ] 了解工业通信协议概念：Modbus、OPC UA、MQTT、Profinet

### 🟨 嵌入式/编程

- [ ] STM32 GPIO/Timer/PWM/中断/ADC/串口/I2C/SPI 全部用过
- [ ] 独立完成过一个带闭环控制的 STM32 项目
- [ ] 能用 Python 写数据处理脚本（pandas + matplotlib）
- [ ] 会用 Git/GitHub 管理代码
- [ ] 了解数据库基础（会用 SQL 写查询）

### 🔧 系统/集成

- [ ] 理解 MES/ERP 的数据流：工单→报工→质检→入库
- [ ] 了解数字孪生基本概念和技术栈
- [ ] 了解智能制造典型场景（工信部参考指引）

### 🗣 软技能

- [ ] 能用结构化思维写技术文档
- [ ] 能对着电路图/梯形图/代码讲清楚自己的项目
- [ ] CET-4 必须过，CET-6 尽量过
